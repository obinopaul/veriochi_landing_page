# Veriochi - The AI Workspace

**Your AI. Your Studios. One Workspace.**

Veriochi is a next-generation cloud workspace designed to unify your files, tools, and AI assistants into one intelligent platform. By providing specialized **AI Studios** for different creative and technical tasks, Veriochi empowers you to build, write, design, and code without switching contexts.

## üõ†Ô∏è Getting Started

### Prerequisites
- Node.js 18+
- pnpm (recommended) or npm/yarn

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/veriochi/veriochi_landing_page.git
    cd veriochi_landing_page
    ```

2.  **Install dependencies:**
    ```bash
    pnpm install
    ```

3.  **Run the development server:**
    ```bash
    pnpm dev
    ```

    Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
